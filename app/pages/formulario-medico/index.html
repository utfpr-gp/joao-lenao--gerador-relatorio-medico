Gerador de Relatório Médico
Legibilidade 100% garantida!
Valdisney é médico em um hospital de Nova Iorque, uma cidade do interior do Maranhão. Neste hospital também trabalha João Lenão, um médico plantonista com ortografia praticamente inelegível. Como os relatórios dos pacientes sempre são escritos à mão, Valdisney sempre tem problemas em ler os relatórios escritos por João Lenão. Nestes dias, até o próprio João Lenão relatou problemas com a leitura de seus relatórios. Neste sentido, Valdisney pensou em criar um script para gerar o relatório a partir de informações sobre o paciente inseridas em um formulário. Porém, ele não tem tanto conhecimento de JavaScript e ultimamente está muito atarefado com as atividades do hospital e interpretação dos relatórios de João Lenão. Desta forma, faça uma boa ação, crie este script para Valdisney!

Exigências:

  • Definir uma classe Relatorio e uma classe Usuario e instanciar um objeto de cada classe para conter os dados inseridos no formulário. Os dados destes objetos precisam ser usados para imprimir o relatório formatado logo abaixo do formulário.
  • O texto formatado com os valores do formulário devem ser apresentados dentro de um DIV.
    • Para apresentação, as datas precisam ser formatadas no formato DD/MM/YYYY dentro de uma função com nome e usando String Template. Vale salientar que a data de nascimento é lida como string em JavaScript e deve ser manipulada como tal.
    • Para apresentação dos sintomas, transformar os dados do vetor de sintomas em string (separando cada sintoma por vírgula) via um método nativo da classe Array.
